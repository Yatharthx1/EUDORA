# Smart Traffic Routing System

This project is a city-scale traffic routing system built using real road network data of Indore via OpenStreetMap and OSMnx.

The system currently computes the fastest route between two locations based on estimated travel time derived from road length and assumed average speed.

## Upcoming Additions

We aim to extend the routing engine by introducing two additional path types:

1. **Smoothest Route**
   A path optimized to reduce expected red-light stops and waiting time using simulated signal cycle modeling.

2. **Least Pollution Route**
   A path optimized to minimize pollution exposure by reducing idle time and stop frequency.

The system is fully simulation-based and does not control real-world traffic infrastructure.
